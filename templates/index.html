TDI Capstone Project- Identifying Secret Ingredients

Enter a a dish, get some science.

Item:

This project is concerned with identifying what I'm calling secret ingredients. They're not secrets because nobody knows what they are, but they're secrets because you gotta try a hundred different recipes before you know which ingredients make a difference. That's where data science comes in.

This project utilizes the Yummly API, analyzing ratings and ingredients for each recipe.

Currently, you can search by dish, and an ingredient breakdown and discussion is provided.
